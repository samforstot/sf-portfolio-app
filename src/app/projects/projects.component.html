Mealdealz

Mealdealz is a webapp that streamlines cooking and grocery shopping in the attempt to make the lives of college students (or anyone) easier! For this project, I collaborated with a team using an agile workflow and HTML, CSS, and Typescript within the Angular Frameowork. I also wrote Python scripts that run within a VM periodically to scrape weekly deals from various grocery stores.

Gator Path Tracker

The Gator Path Tracker (GPT) is an in-development project makes use of a web driver within various Python scripts to grab student-produced opinions and data relating to the overall difficulty of classes at the Univeristy of Florida. Sources include sites ranging from Reddit (r/ufl) to RateMyProfessor. This data is then passed to OpenAI's ChatGPT API to perform language processing to rate each class and further build a schedule for users. Click here to check it out!

Yappin

Yappin! is a web app made with the MERN stack and web sockets. It allows users to join individual groups and develop quote walls with friends. Quotes can be viewed and liked in real time, as each group acts as an instant messaging groupchat! User authentication is also implemented using jwts. This project was mainly used for a first attempt to use REACT and Mongodb to get a better feel for the fundamentals of each. Click here to check it out!

D.R.I.G.

The Direct Replay Information Gatherer is an application that displays information based on around 300,000 individual statlines from the National Basketball Association from 2019-2022. This project is meant to test the varying time differences when executing quick sort and merge sort algorithms. Using C++, Windows Forms .NET Framework, and Adobe Photoshop, a UI was created to display results of user searches. Click here for repo!
